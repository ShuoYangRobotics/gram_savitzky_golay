ContainerT::value_type gram_sg::SavitzkyGolayFilter::filter ( const ContainerT &  v) const

Apply Savitzky-Golay convolution to the data x should have size 2*m+1 As the function only applies a convolution, it runs in O(2m+1), and should be rather fast.

Parameters
vContainer with the data to be filtered. Should have 2*m+1 elements Type of elements needs to be compatible with multiplication by a scalar, and addition with itself Common types would be std::vector<double>, std::vector<Eigen::VectorXd>, boost::circular_buffer<Eigen::Vector3d>...
Returns
Filtered value according to the precomputed filter weights.
